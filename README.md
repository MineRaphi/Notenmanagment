# Notenmanagement

A mobile App for the HTL-Braunau Notenmanagement

## Description

Mobile Notenmanagement App using the HTL-Braunau Notenmanagement Rest API.
Uses Capazitor and Ionic

## Getting Started

### Dependencies

* [Visual Studio Code](https://code.visualstudio.com)
* [NodeJS](https://nodejs.org/en/download)
* [XCode](https://developer.apple.com/xcode/) (for IOS)
* [Android Studio](https://developer.android.com/studio) (for Android)

### Installing

* Download source files from the repo
* Install the Node packages

### Executing program

* Install/Update packages
```
npm install
```

* Building for IOS
```
npm run build-ios
```

* Build for Android
```
npm run build-android
```
